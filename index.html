Quase lá

O painel busca pôsteres, elenco e gêneros dos seus filmes no TMDB. Cole sua chave gratuita uma vez — fica só no seu aparelho.

Como pego a chave?
  1. Crie conta em themoviedb.org.
  2. Vá em Configurações → API.
  3. Copie a Chave da API (v3) — a curta.

Ícone da tela inicial

Dados e imagens via TMDB; produto não endossado pelo TMDB. Atualizar filmes mescla o novo export com o que já existe (não apaga histórico).

"Salvar HTML" gera um novo arquivo com todos os seus dados embutidos — faça commit no GitHub para garantir backup permanente.

Avaliações e dados ficam no IndexedDB do navegador. Ao salvar o HTML e fazer commit, os dados ficam gravados no próprio arquivo — nada se perde ao abrir em nova aba ou novo aparelho.
